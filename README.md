**Requirements**:
- nodejs
- express
- elasticsearch

**Install app**:
1. `git clone <this project>`
2. `yarn`

**Run app**:
`npm run dev`

**Run ES (docker)**:

0. Install docker and docker-compose
1. `cd docker`
2. `docker-compose up -d`
